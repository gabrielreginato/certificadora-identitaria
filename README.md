# Projeto - Certificadora de Competência Identitária

## Sistema Proposto
<p>Sistema web que visa centralizar a gestão de oficinas acadêmicas por meio de uma aplicação Full-stack que permite ao usuário cadastrar, consultar e editar informações de oficinas (datas e horários de encontros, temas e descrições), bem como professores, tutores e alunos vinculados.</p>

## Tecnologias Utilizadas
<p>A aplicação Back-end será composta por uma API Rest desenvolvida em Node.js, integrada a um banco de dados MySQL. Já o Front-end será desenvolvido em React.js, para facilitar a criação de uma UI responsiva e manutenível.</p>

## Requisitos Funcionais

| ID   |                                 Requisito                                 |
| :--: | :-----------------------------------------------------------------------: | 
| RF1 |  O sistema deve permitir a criação de oficinas contendo as seguintes informações: <ul> <li>Professores vinculados;</li> <li>Tutores vinculados;</li> <li>Tema;</li> <li>Descrição;</li> <li>Encontros (um ou mais), contendo: <ul> <li>Data;</li> <li>Horário;</li> </ul> </li></ul>|
| RF2 |  O sistema deve permitir o cadastro e gerenciamento de alunos, professores e tutores.|
| RF3 | O sistema deve permitir a vinculação de alunos, professores e tutores às oficinas.|
| RF4 |  O sistema deve permitir a edição dos registros existentes.|
| RF5 | O sistema deve permitir a exclusão dos registros existentes.|
| RF6 |  O sistema deve permitir a visualização dos registros através de uma interface visual.|
