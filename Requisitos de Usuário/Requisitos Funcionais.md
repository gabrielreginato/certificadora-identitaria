| ID   |                                 Requisito                                 |
| :--: | :-----------------------------------------------------------------------: | 
| RF1 |  O sistema deve permitir a criação de oficinas contendo as seguintes informações: <ul> <li>Professor responsável;</li> <li>Descrição;</li> <li>Encontros (um ou mais), contendo: <ul> <li>Data;</li> <li>Horário;</li> </ul> </li></ul>|
| RF2 |  O sistema deve permitir o cadastro e gerenciamento de alunos e professores.|
| RF3 | O sistema deve permitir a vinculação de alunos (como participantes), professores (como tutores) às oficinas.|
| RF4 |  O sistema deve permitir a edição dos dados das oficinas existentes.|
| RF5 | O sistema deve permitir a exclusão dos registros (de oficinas e vínculos) existentes.|
| RF6 |  O sistema deve permitir a visualização dos registros (de oficinas e vínculos) e informações do usuário através de uma interface visual.|
